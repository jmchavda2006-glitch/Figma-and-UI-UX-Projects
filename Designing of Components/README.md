### 🚀 Project Overview
This repository focuses on the recreation and evolution of iconic software components. I’ve taken standard productivity icons (Microsoft Office Suite) and system elements (Windows 11) and reimagined them using **Glassmorphism**—focusing on background blur, multi-layered transparency, and realistic highlights.

</div>

## 🎨 Design Modules

### 1️⃣ Microsoft Productivity Suite (Standard vs. Glass)
A side-by-side comparison of the classic Office icons and their glass-textured counterparts.
*   **Icons:** Word, Excel, and PowerPoint.
*   **The Challenge:** Maintaining the recognizable brand colors (Blue, Green, Orange) while applying translucent depth and inner glows.

### 2️⃣ System Utility Components
Custom-built productivity tools designed with high-end UI aesthetics.
*   **Calendar & Notes:** Hand-crafted vector icons featuring "frosted glass" panels and dynamic drop shadows.
*   **Visuals:** Each component is built using layered shapes to simulate light refraction.

### 3️⃣ Windows 11 Ecosystem
Recreating the modern Fluent Design system from scratch.
*   **Start Button:** A pixel-perfect recreation of the Windows 11 logo using precise geometry.
*   **Transparent Glass Interface:** A standalone exploration of a 100% transparent glass component, showcasing how UI elements interact with varying backgrounds.

## 💎 Technical Highlights

<table>
  <tr>
    <td><b>Layered Transparency</b></td>
    <td>Using multiple fill layers (solid, linear, and radial gradients) to mimic how real glass reflects light.</td>
  </tr>
  <tr>
    <td><b>Vector Fidelity</b></td>
    <td>Every icon was built using the Pen Tool—no PNGs or external assets. This allows for infinite scaling without quality loss.</td>
  </tr>
  <tr>
    <td><b>Blur Logic</b></td>
    <td>Careful application of Background Blur (ranging from 10px to 40px) to ensure readability against complex wallpapers.</td>
  </tr>
</table>

### 🚀 Overview
Welcome to my design repository. This space showcases my journey in **Frontend Engineering** and **UI/UX Design**, focusing on building complex interfaces from the ground up. Every icon and component found here is a result of manual vector work—zero pre-made kits used.

## 📂 Repository Structure

This repository is organized into three core modules, each exploring different aspects of modern UI design:

### 📱 [Samsung One UI](./one%20ui)
**High-Fidelity Mobile Ecosystem Recreation**
*   **Focus:** Layout precision, system-wide consistency, and custom status bar architecture.
*   **Highlights:** Hand-crafted battery and network icons, Quick Panel animations, and Light/Dark mode transitions.

### 🛒 [Tag Kart](./tag%20kart)
**Product Ecosystem & Marketing Design**
*   **Focus:** User onboarding (Login UX), digital advertising, and informational layouts.
*   **Highlights:** Glassmorphic login cards, high-impact ad banners, and multi-page brochure systems.

### 💎 [Components](./components)
**Advanced UI Element Library**
*   **Focus:** Material studies in Glassmorphism and Fluent Design.
*   **Highlights:** Reimagined Microsoft Office icons, Windows 11 system elements, and custom productivity widgets (Calendar/Notes).

---

## 🛠️ Design Philosophy

| Principle | Execution |
| :--- | :--- |
| **Zero Assets** | Every icon is built using raw shapes and Boolean operations. |
| **Logic First** | Designs are built with frontend implementation and user flow in mind. |
| **Material Depth** | Extensive use of background blurs, inner glows, and shadow layering. |

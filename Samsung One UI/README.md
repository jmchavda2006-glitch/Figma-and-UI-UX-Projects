### 🚀 Project Overview
This repository showcases a meticulous recreation of the **Samsung One UI** interface. While most designers use pre-made kits, this project is a pure exercise in precision—**every single system icon and UI component was manually constructed** using raw vector shapes.

</div>

## 📱 Interactive Modules

### 1️⃣ Home Screen & Quick Panel
Focused on the core navigation and the complex notification architecture.
*   **Visuals:** Custom status bar and interactive Quick Settings tray.
*   **Logic:** Smooth pull-down animations and icon scaling.
*   [**🔗 Open Figma Project**](https://www.figma.com/design/noPRpksBQSVRVkw3DJYHF9/One-UI-Home-and-Quick-Panel?node-id=0-1&t=Wm8ivo9RyyN718c0-1)

### 2️⃣ Light & Dark Mode Switching
An exploration of color theory and interface adaptability.
*   **Visuals:** Side-by-side comparison of theme transitions.
*   **Logic:** How hand-built icons respond to contrast shifts.
*   [**🔗 Open Figma Project**](https://www.figma.com/design/yzRoxD8NAP4HcueGlOnqzh/Light-Dark-Mode?node-id=0-1&t=8f8dHGSBMwzFKuUf-1)

---

## 🛠️ Technical Details
*   **Software:** Figma (Pen Tool, Boolean Operations, Corner Smoothing).
*   **Assets:** 100% original vector work.
*   **Backgrounds:** Wallpapers sourced from Google Images for realistic context.
